+# 홈택스 신용카드 매입내역 자동 다운로드
+
+아이디/비밀번호 로그인 후, 원하는 기간의 **사업용 신용카드 매입내역**을 조회하고 **엑셀 파일**로 저장하는 스크립트입니다.
+
+> 홈택스 UI/보안 정책은 자주 바뀝니다. 간편인증·공동인증서가 나오면 브라우저에서 직접 완료해야 합니다.
+
+## 1. 설치
+
+```bash
+python -m venv .venv
+source .venv/bin/activate
+pip install playwright
+playwright install chromium
+```
+
+## 2. 실행
+
+```bash
+python hometax_card_downloader.py <아이디> <비밀번호> --from-date 2024-01-01 --to-date 2024-03-31
+```
+
+비밀번호를 커맨드라인에 남기기 싫다면 생략 가능(프롬프트 입력):
+
+```bash
+python hometax_card_downloader.py <아이디> --from-date 2024-01-01 --to-date 2024-03-31
+```
+
+옵션:
+
+- `--download-dir ./downloads`: 저장 폴더
+- `--headless`: 창 없이 실행
+- `--manual-auth-timeout 300`: 추가 인증 대기 시간(초)
+- `--keep-open-on-fail`: 오류 시 브라우저를 닫지 않고 유지(디버깅용)
+
+## 3. 실제 동작 흐름
+
+1. 로그인 페이지 접속
+2. 아이디/비밀번호 입력 및 로그인 클릭
+3. 추가 인증 완료까지 대기
+4. 사업용 신용카드 매입내역(FN0206) 페이지 이동
+5. 기간(시작/종료일) 입력
+6. 조회 클릭
+7. 엑셀 다운로드
+
+## 4. 안 될 때 점검
+
+- 로그인 직후 추가 인증이 필요한 경우: 인증 완료까지 기다려야 다음 단계 진행
+- 조회/엑셀 버튼을 못 찾는 경우: 홈택스 화면 구조 변경 가능성 높음
+- 다운로드가 안 되는 경우: 팝업/보안모듈 영향 가능
+
+필요 시 아래 상수 목록의 선택자를 최신 화면 기준으로 보강하세요.
+
+- `DATE_FROM_SELECTORS`
+- `DATE_TO_SELECTORS`
+- `SEARCH_SELECTORS`
+- `EXCEL_SELECTORS`
+
+## 5. 보안 주의
+
+- 비밀번호를 CLI 인자로 전달하면 쉘 히스토리에 남을 수 있습니다.
+- 운영 환경에서는 프롬프트 입력 또는 환경변수/시크릿 관리 사용을 권장합니다.
