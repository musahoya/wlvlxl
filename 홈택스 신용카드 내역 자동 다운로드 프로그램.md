# 홈택스 신용카드 내역 자동 다운로드 프로그램

이 프로그램은 Python과 Playwright 라이브러리를 사용하여 국세청 홈택스에 자동으로 로그인하고 사업용 신용카드 매입 내역 페이지까지 이동하는 기능을 제공합니다.

## 1. 사전 준비 사항

이 프로그램을 실행하기 위해서는 Python이 설치되어 있어야 하며, 다음과 같은 라이브러리 설치가 필요합니다.

```bash
# 필요한 라이브러리 설치
pip install playwright

# 브라우저 드라이버 설치
playwright install chromium
```

## 2. 사용 방법

터미널(또는 명령 프롬프트)에서 다음과 같이 실행합니다.

```bash
python hometax_card_downloader.py [사용자_아이디] [사용자_비밀번호]
```

## 3. 주요 기능 및 참고 사항

*   **자동 로그인**: 입력한 아이디와 비밀번호를 사용하여 홈택스 로그인 과정을 자동화합니다.
*   **메뉴 이동**: 로그인 후 '사업용 신용카드 매입내역 조회' 페이지로 직접 이동합니다.
*   **보안 관련 안내**: 
    *   홈택스는 보안 프로그램 설치를 강력하게 권장하며, 자동화 도구(Bot)의 접근을 차단할 수 있습니다.
    *   아이디/비밀번호 로그인 후에도 **추가 인증(간편인증, 공동인증서 등)**이 필요한 경우, 프로그램이 대기 상태일 때 사용자가 직접 브라우저에서 인증을 완료해야 할 수 있습니다.
    *   스크립트 내 `headless=False` 설정은 브라우저 창을 띄워 동작 과정을 직접 볼 수 있게 해줍니다.

## 4. 커스터마이징

홈택스의 웹사이트 구조는 수시로 변경될 수 있습니다. 만약 버튼 클릭이나 페이지 이동이 정상적으로 동작하지 않는다면, 스크립트 내의 `Selector` (예: `#mf_txppWframe_anchor25`)를 최신 구조에 맞게 수정해야 합니다.
